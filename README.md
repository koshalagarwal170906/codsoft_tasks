# CodSoft Java Development Internship

This repository contains all the Java programming tasks completed as part of the **CodSoft Virtual Internship Program**.

---

## 📁 Project Structure & Tasks

All task implementation files are located inside the `src/` directory.

| Task # | Task Name | Source File | Description |
| :--- | :--- | :--- | :--- |
| **Task 1** | Number Game | [`src/NumberGame.java`](./src/) | A console-based number guessing game with limited attempts and score tracking. |
| **Task 2** | Student Grade Calculator | [`src/GradeCalculator.java`](./src/) | Calculates total marks, average percentage, and assigns grades based on input marks. |
| **Task 3** | ATM Interface | [`src/AtmInterface.java`](./src/) | A simple ATM machine system supporting deposit, withdrawal, and balance check. |

*(Note: Aap apne actual filenames ke hisab se text change kar sakte hain)*
